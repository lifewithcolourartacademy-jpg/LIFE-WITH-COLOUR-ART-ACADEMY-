## 📁 Project Structure

- `index.html` – Main HTML file for the homepage
- `.jpg` & `.png` files – Images used on the website
- `README.md` – You're reading it right now!

---

## ✏️ How to Update the Site

1. Go to the repository
2. Click on the `index.html` file
3. Click on the ✏️ **Edit this file** button
4. Make your changes in the HTML code
5. Scroll down and click **Commit changes**

Your live website will be updated within 1–2 minutes!
